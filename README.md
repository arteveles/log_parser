# log_parser

1. Сперва рекомендую отыскать файл с логами в ручном режиме, чтоб в п.2 вставить путь до файла
команда: locate access.log
2. Запускать парсер командой: python parser.py -f ПУТЬ_ИЗ_ШАГА_1
